# 🌸 Pibot

Pibot es un bot de Discord enfocado en el entretenimiento social.
Convierte tu servidor en una experiencia interactiva completa con sistemas avanzados y diversión constante.

</p>

---

## ✨ Características

- 💰 Economía virtual avanzada
- 🛒 Tienda, ítems y sistema de crafteo
- 🏆 Misiones, logros y eventos
- 🎮 Minijuegos, apuestas y tradeos
- 📈 Sistema de niveles
- 🎵 Reproductor de música
- 🤖 Chat con IA integrado
- ⚙️ Sistema de configuración por servidor

---

## 🧠 Tecnologías

- Node.js
- Discord.js
- Base de datos persistente
- API de Groq (IA)
- Lavalink (Sistema de música)
