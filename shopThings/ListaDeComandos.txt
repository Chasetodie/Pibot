# 🤖 Sistema Completo de Tienda y Economía - Lista de Comandos

## 💰 **Comandos de Economía Básica**
- `>balance` / `>bal` - Ver tu dinero actual
- `>work` - Trabajar para ganar dinero (con efectos de items)
- `>daily` - Recompensa diaria (bonificación VIP disponible)
- `>gamble <cantidad>` / `>bet <cantidad>` - Apostar dinero
- `>transfer @usuario <cantidad>` / `>pay @usuario <cantidad>` - Transferir dinero
- `>profile` / `>perfil` [@usuario] - Ver perfil (con items equipados y efectos)

## 🏪 **Comandos de Tienda**
- `>shop` / `>tienda` - Ver la tienda completa
- `>shop <categoría>` - Ver categoría específica (consumable, permanent, cosmetic, mystery)
- `>buy <item_id> [cantidad]` / `>comprar <item_id> [cantidad]` - Comprar items
- `>bag` / `>inventario` / `>inv` [@usuario] - Ver inventario
- `>useitem <item_id>` / `>usar <item_id>` - Usar un item

## 🎯 **Comandos de Misiones**
- `>missions` / `>misiones` [@usuario] - Ver misiones diarias
- Las misiones se completan automáticamente al realizar actividades

## 🏆 **Comandos de Logros**
- `>achievements` / `>logros` - Ver tus logros desbloqueados

## 🔄 **Comandos de Intercambio**
- `>trade @usuario item_id:cantidad` - Iniciar intercambio
- `>tradeadd <item_id> [cantidad]` - Agregar item al intercambio
- `>tradeaccept` - Aceptar el intercambio
- `>tradecancel` - Cancelar intercambio

## 🔨 **Comandos de Subastas**
- `>auction <item_id> <precio_inicial> [minutos]` - Crear subasta
- `>bid <auction_id> <cantidad>` - Pujar en una subasta
- `>auctions` - Ver subastas activas

## ⚒️ **Comandos de Crafteo**
- `>recipes` - Ver recetas de crafteo disponibles
- `>craft <recipe_id>` - Craftear un item

## 👑 **Comandos VIP**
- `>vip` - Ver comandos VIP disponibles (requiere Pase VIP)
- `>vipwork` - Trabajo especial VIP
- `>vipbonus` - Bonificación adicional cada 12 horas
- `>vipstats` - Estadísticas detalladas
- `>vipshop` - Items exclusivos VIP

## 🛠️ **Comandos Administrativos** (Solo Administradores)
- `>giveitem @usuario <item_id> [cantidad]` - Dar items a usuarios
- `>givemoney @usuario <cantidad>` - Dar dinero a usuarios
- `>shopstats` - Ver estadísticas de la tienda
- `>resetuser @usuario` - Resetear completamente un usuario

## 📊 **Características del Sistema**

### ⚡ **Efectos de Items**
- **Multiplicadores**: Aumentan ganancias de dinero y XP
- **Reducción de Cooldown**: Menos tiempo de espera en comandos
- **Boost de Éxito**: Mayor probabilidad en actividades
- **Efectos Permanentes**: Mejoras que duran para siempre
- **Efectos por Usos**: Items que se consumen gradualmente

### 🎯 **Sistema de Misiones**
- **5 misiones diarias** que se resetean a medianoche
- **Diferentes tipos**: mensajes, trabajo, dinero ganado, juegos, etc.
- **Rareza de misiones**: Common, Uncommon, Rare, Epic
- **Recompensas**: Dinero y XP que se ven afectados por eventos

### 🏆 **Sistema de Logros**
- **Coleccionista**: Poseer 10 items diferentes
- **Gran Gastador**: Gastar 50,000 π-b$ en la tienda
- **Miembro VIP**: Comprar el Pase VIP
- Más logros se desbloquean automáticamente

### 🔄 **Funciones Avanzadas**
- **Intercambios**: Comercia items directamente con otros usuarios
- **Subastas**: Vende tus items al mejor postor
- **Crafteo**: Combina items para crear otros más poderosos
- **Eventos**: Bonificaciones especiales los fines de semana

## 🗃️ **Instalación y Configuración**

### 1. **Base de Datos**
```sql
-- Agregar a tu tabla users:
ALTER TABLE users 
ADD COLUMN IF NOT EXISTS activeEffects JSON DEFAULT NULL,
ADD COLUMN IF NOT EXISTS permanentEffects JSON DEFAULT NULL;
```

### 2. **Dependencias NPM**
```bash
npm install node-cron
```

### 3. **Variables de Configuración**
- Cambiar `'TU_TOKEN_DEL_BOT'` por tu token real
- Cambiar `'TU_CANAL_ANUNCIOS_ID'` por el ID de tu canal de anuncios
- Configurar tu base de datos en el archivo de economía

### 4. **Estructura de Archivos**
```
📁 tu-bot/
├── 📄 index.js (archivo principal)
├── 📄 economy.js (tu sistema de economía existente)
├── 📄 shop.js (sistema de tienda)
├── 📄 missions.js (tu sistema de misiones existente)
├── 📄 advanced-features.js (características avanzadas)
└── 📄 events.js (sistema de eventos, opcional)
```
