POLÍTICA DE PRIVACIDAD – PIBOT

Última actualización: 03/03/26

1. Datos Recopilados

Pibot almacena únicamente los datos necesarios para su funcionamiento, incluyendo:

- ID de usuario de Discord
- ID de servidor
- Datos de economía (dinero virtual, inventario, progreso)
- Configuración del servidor
- Estadísticas de uso

Pibot NO almacena contraseñas ni información personal sensible.

2. Uso de los Datos

Los datos se utilizan exclusivamente para:
- Mantener el progreso del usuario
- Gestionar la economía virtual
- Guardar configuraciones
- Permitir el funcionamiento de sistemas como misiones, logros y eventos

3. IA y Mensajes

Cuando un usuario utiliza el chat con IA:
- El mensaje puede enviarse a la API de Groq para generar una respuesta.
- No se almacenan conversaciones completas de forma permanente, salvo que sea necesario para funcionamiento técnico.

4. Música

El sistema de música utiliza Lavalink, que puede conectarse a fuentes externas.
Pibot no almacena contenido musical ni archivos de audio.

5. Compartición de Datos

Pibot NO vende, alquila ni comparte datos con terceros.
Los datos solo se usan internamente para el funcionamiento del bot.

6. Eliminación de Datos

Los usuarios pueden solicitar la eliminación de sus datos contactando al desarrollador.
El desarrollador se reserva el derecho de eliminar datos inactivos.

7. Seguridad

Se aplican medidas razonables para proteger los datos almacenados.
Sin embargo, ningún sistema es 100% seguro.
